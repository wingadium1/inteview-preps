# Interview Preparation Repository

Welcome to the comprehensive interview preparation repository! This repository contains structured preparation materials for key technologies and concepts commonly tested in software engineering interviews.

## üìö Topics Covered

### 1. [Java](./Java/README.md)
Core Java concepts, collections, multithreading, JVM, design patterns, and modern Java features (Java 8+).

**Key Topics:**
- Object-Oriented Programming
- Collections Framework
- Exception Handling
- Multithreading & Concurrency
- Java 8+ Features (Streams, Lambda, Optional)
- Memory Management & Garbage Collection
- Design Patterns

### 2. [Spring Framework](./Spring/README.md)
Spring ecosystem including Spring Boot, Spring MVC, Spring Data JPA, Spring Security, and Spring AOP.

**Key Topics:**
- Dependency Injection & IoC
- Spring Boot & Auto-configuration
- Spring MVC & REST APIs
- Spring Data JPA
- Spring Security
- Spring AOP
- Transaction Management

### 3. [Hibernate](./Hibernate/README.md)
Object-Relational Mapping (ORM) with Hibernate, entity mapping, HQL, and performance optimization.

**Key Topics:**
- ORM Concepts
- Entity Mapping & Relationships
- Hibernate Session Lifecycle
- HQL & Criteria API
- Caching (First-level, Second-level)
- Performance Optimization
- Transaction Management

### 4. [Cloud Computing](./CloudComputing/README.md)
Cloud computing fundamentals, distributed computing platforms, and cloud service models across major providers.

**Key Topics:**
- Cloud Computing Fundamentals
- Distributed Computing Platforms
- Elastic Compute vs Bare Metal Compute
- Cloud Service Models (IaaS, PaaS, SaaS)
- Compute Services (VMs, Serverless, Containers)
- Storage Services (Object, Block, File Storage)
- Database Services (SQL, NoSQL)
- Networking & Security
- Multi-cloud Architecture

### 5. [DevOps](./DevOps/README.md)
DevOps practices, CI/CD pipelines, Infrastructure as Code, and automation.

**Key Topics:**
- Version Control (Git)
- CI/CD (Jenkins, GitHub Actions, GitLab CI)
- Infrastructure as Code (Terraform, Ansible, CloudFormation)
- Configuration Management
- Monitoring & Logging
- Scripting & Automation
- DevSecOps

### 6. [Containers](./Containers/README.md)
Containerization with Docker and orchestration with Kubernetes.

**Key Topics:**
- Docker (Images, Containers, Dockerfile)
- Docker Compose
- Kubernetes Architecture
- Kubernetes Objects (Pods, Deployments, Services)
- kubectl Commands
- Container Best Practices

### 7. [System Design](./SystemDesign/README.md)
System design principles, scalability, distributed systems, and common design patterns.

**Key Topics:**
- Scalability (Vertical & Horizontal)
- Load Balancing
- Caching Strategies
- Database Design & Scaling
- CAP Theorem
- Message Queues
- API Design
- Microservices Architecture

### 8. [Redis](./Redis/README.md)
In-memory data store, caching strategies, and Redis data structures.

**Key Topics:**
- Redis Data Structures (Strings, Lists, Sets, Hashes)
- Persistence (RDB, AOF)
- Replication & High Availability
- Redis Sentinel & Cluster
- Pub/Sub
- Common Use Cases (Caching, Session Storage, Rate Limiting)

### 9. [Microservices](./Microservices/README.md)
Microservices architecture, patterns, communication, and best practices.

**Key Topics:**
- Microservices Principles
- Service Communication (REST, gRPC, Message Queues)
- Service Discovery
- API Gateway
- Circuit Breaker Pattern
- Data Management (Saga Pattern, CQRS)
- Security & Monitoring

## üéØ How to Use This Repository

### For Interview Preparation
1. **Start with Fundamentals**: Begin with Java and Spring if you're preparing for backend roles
2. **Build on Core Concepts**: Move to Hibernate for database interactions
3. **Learn Cloud & DevOps**: Study AWS and DevOps for modern application deployment
4. **Master Containers**: Understand Docker and Kubernetes for containerized applications
5. **Study System Design**: Prepare for system design interviews
6. **Deep Dive into Specialized Topics**: Redis and Microservices for advanced roles

#### Week 7-8: Containers & Microservices
- Day 43-45: Docker fundamentals
